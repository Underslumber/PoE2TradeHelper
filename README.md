## Цель

Сделать локального помощника для торговли в Path of Exile 2, который помогает быстро находить потенциально выгодные сделки, обмены и предметы для перепродажи.

## Хотим получить

- Выбор актуальной PoE2-лиги и работа с ее рыночными данными.
- Сбор текущих цен по валютам, фрагментам, эмоциям, эссенциям, рунам и другим полезным категориям.
- Нормализация цен из разных источников, чтобы можно было сравнивать позиции между собой.
- История снимков цен, чтобы видеть изменение рынка во времени.
- Поиск обменов с потенциальной прибылью с учетом spread, объема, свежести listings и риска price fixing.
- Анализ 3-в-1 цепочек, например Liquid Emotions и других категорий, где есть апгрейд/обмен.
- Отдельный анализ проходок, фрагментов и boss-entry предметов.
- Фильтр сигналов по практической исполнимости: не только математическая прибыль, но и достаточный объем рынка.
- Удобный локальный интерфейс для просмотра категорий, цен, медиан, объема, изменения за период и найденных сигналов.
- Возможность вставить текст предмета из игры и получить оценку цены или торгового потенциала.
