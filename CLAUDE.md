# CLAUDE.md

Этот файл намеренно короткий. Канонический гид для всех агентов (включая Claude Code) —
[AGENTS.md](AGENTS.md). Прочитай его перед любыми нетривиальными изменениями.

`AGENTS.md` содержит:

- назначение проекта и рабочую базу;
- критичные API-факты по `trade2`, poe.ninja и официальному `api.pathofexile.com`;
- правила разработки (версионирование, i18n, rate limit, торговые выводы, ИИ, FunPay);
- команды проверки (`py_compile`, `pytest`, live-проверка справочников);
- правила работы с файлами и секретами;
- правила коммуникации.

Если правишь подход к API или поведение проекта — обновляй `AGENTS.md` (и `README.md`),
а не дублируй содержимое сюда.
